FormMirror - 動作分析・フォームチェック Webアプリ

概要

FormMirror は、体育の授業や部活動、個人のスポーツトレーニングで手軽に利用できる動作分析アプリです。
ブラウザだけで動作し、自分の動きを撮影して「お手本動画」と比較したり、AIによる骨格検知機能でフォームを視覚的に分析することができます。

主な機能

1. 2画面比較 & 同時再生

お手本動画: 先生やプロ選手の動画ファイルを読み込み、左側に表示できます。

自分の動き: カメラで撮影した動画を右側に表示します。

同時コントロール: 「再生」「一時停止」「リセット」ボタンで2つの動画を完全に同期して操作し、タイミングのズレを確認できます。

2. AI動作分析 (MediaPipe Pose)

骨格表示: 体の関節位置をAIがリアルタイムに検出し、線で結んで表示します。体の軸や姿勢の確認に最適です。

軌跡表示: 手首や足首の動きをトレースし、動いた軌跡を線で残します。スイングや投球フォームのブレを可視化します。

表情ワイプ (Face Zoom): 全身を映していても表情がわかるよう、顔部分を自動で検出・拡大して画面隅に表示します。

3. 使いやすい録画機能

カウントダウン: 録画ボタンを押してから3秒間の待機時間があり、定位置についてから撮影を開始できます。

ミラーモード: 鏡のように左右反転して表示するため、直感的に動きを確認できます。

スロー再生: 0.25倍、0.5倍のスロー再生に対応しており、速い動きも細かくチェック可能です。

4. レスポンシブデザイン

スマートフォン、タブレット、PCのいずれでも最適なレイアウトで表示されます。

「表示切替」ボタンで、比較なしのシングルモードにも切り替え可能です。

使い方

準備:

アプリを開き、「カメラを起動」ボタンを押してブラウザのカメラ利用を許可します。

比較したい場合、「動画を選択」からお手本となる動画ファイルを読み込みます。

撮影:

「録画開始」ボタンを押します。3秒のカウントダウン中に位置につきます。

動作を行い、「録画終了」ボタンを押します。

分析:

自動的にプレビューモードになります。

「再生」ボタンで動きを確認したり、画面上部のチェックボックスで「骨格表示」や「軌跡表示」をオン/オフして分析します。

納得いくまで「撮り直す」ボタンで何度でもトライできます。

技術スタック

Styling: Tailwind CSS (CDN)

AI/ML: MediaPipe Pose (Google)

Icon: FontAwesome

プライバシーについて

このアプリはクライアントサイド（ブラウザ内）ですべての処理が完結します。撮影した動画データやカメラ映像が外部サーバーに送信・保存されることはありません。安心してご利用ください。
